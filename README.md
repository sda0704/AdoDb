Задание:  
Написать консольную утилиту, реализующую CRUD запросы к БД через:  

ADO.NET  

Адрес сервера СУБД должен задаваться в строке подключения из конфига консольной утилиты встроенными средствами.  

Стек технологий:  

C# / .NET
ADO.NET

Структура проекта:

1. Program.cs - Главное меню и взимодействие с пользователем
2. Configuration.cs - Работа с JSON конфигурацией. Поиск JSON файла  
3. User.cs - Модель данных
4. UserRepository - CRUD операции с базой данных
5. appsettings.json - Строка подключения


Функциональность:  
1. Создание записи в таблице  
2. Обновление записи в таблице  
3. Поиск записи в таблице  
4. Удаление записи из таблицы  
5. Вывод всех записей таблицы
